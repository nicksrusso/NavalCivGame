# Game

## Goals

This is a strategy game inspired by Sid Meyer's civilization games and the Pacific Theater in WWII. Its a sandbox to implement a Gymnasium/petting zoo compatible environment to test AI agents game playing. 

The game has 2 players and a map consisting of water, land, and ports. A player wins by capturing the enemy home port. Turns are taken sequentially, with each player allowed to choose as many or few moves as desired per turn. 

## Combat 

A port can be captured by moving a landing ship onto its tile. Ship vs ship combat is resolved as a binary outcome (ship is alive or dead, no damage model). Each ship type has a cost, strength, attack range, speed, visibility coefficient, and scouting coefficient

A ship can attack any spotted enemy ship within its attack range on a given turn. Ships can move before attacking. When a player attacks, a probability of kill is calculated based on the relative strength of the attacker and defender, as well as a modifier based on each ship's type. The probability distribution is sampled, and the defending ship is deleted if destroyed. 

Detection range for two ships is calculated as the product of each's visibility coefficient and scouting coefficient. High visibility coefficients mean the ship is larger and will be more easily spotted by the enemy, while a higher scouting coefficient increases the detection range 

## Economy

Each turn, every port a player owns generates 25 raw materials. These raw materials are transformed into cash if the materials are transported to the home port by transport ships (capacity 100 raw materials per ship). A player is free to spend cash on new ships or research (to improve ships combat capability) or save each turn. Cash generated by a transport ship landing at the home port is available to spend on the current turn. 

Enemy ports can be captured by placing a landing ship on the tile containing the port. New ports can be built by purchasing a builder ship and moving onto a tile containing empty land. Builder ships are single use and will be deleted once a port is built

## Map

All distances are measured as manhattan distance. A simple yaml config is used to define top level map params, then a map is randomly created and stored in a json
